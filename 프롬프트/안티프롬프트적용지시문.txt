지금 사이트의 기능 로직은 그대로 유지하면서, 서비스 컨셉인 '소중한 기억의 복원'에 맞춰 전체적인 디자인과 톤앤매너를 따뜻하고 감성적으로 리뉴얼하고 싶어. 코드가 꼬이지 않도록 아래 가이드라인을 지켜서 작업해줘.
1. 디자인 시스템 가이드 (Tailwind 설정)
기존의 차가운 사무적 느낌을 벗고, '따뜻한 햇살' 같은 느낌을 주는 컬러와 스타일을 적용해줘.
Color Palette (Custom 설정):
Primary: #FFB7A0 (포인트 살구색)
Background: #FFF8F0 (따뜻한 크림색)
Text: #4A4A4A (부드러운 차콜 그레이)
Style Specs:
border-radius: 24px (모든 카드와 버튼의 모서리를 더 둥글게)
box-shadow: 0 10px 30px rgba(0,0,0,0.05) (부드럽고 옅은 그림자)
2. 감성적인 '사연 포장'을 위한 UI/UX 변경
기능 위주의 영문 문구를 사용자에게 감동을 주는 한글 문구로 교체해줘.
기존 항목	변경할 감성 문구 (Korean)
Create the Photo You Always Dreamed Of.	당신이 꿈꾸던 그 순간을 다시 마주하세요.
Drag your photo here	소중한 추억이 담긴 사진을 올려주세요.
Wedding Again	다시 쓰는 결혼식의 약속
Family Memory	우리 가족의 가장 따뜻한 날
Pet Memory	우리 곁을 지켜준 영원한 친구
Traditional Culture	전통이 깃든 소중한 기록
3. UX/UI 디테일 수정
로딩 상태: 이미지 생성 중 나타나는 로딩 문구를 "기억을 아름답게 복원하고 있어요..."와 같이 감성적인 문구들로 랜덤하게 출력해줘.
아이콘: 전체적으로 날카로운 느낌의 아이콘 대신, 부드러운 라인 아트 스타일의 아이콘을 사용해줘.
레이아웃: 카드 섹션들에 여백을 더 주어 전체적으로 여유롭고 편안한 느낌을 줘.
4. 코드 꼬임 방지 요청사항
기능 유지: 이미지 생성 로직 및 업로드, BYOK 기능 등 기존의 핵심 로직은 절대 건드리지 마.
스타일 분리: 하드코딩하지 말고 tailwind.config.ts에서 위 컬러 팔레트를 변수로 등록해서 일괄 적용해줘.
텍스트 교체: 코드 곳곳에 흩어진 영문 문구들을 위 표의 한글 문구로 교체해줘.
